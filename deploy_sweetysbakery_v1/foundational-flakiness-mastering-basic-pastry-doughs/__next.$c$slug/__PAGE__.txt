Gluten, formed when flour proteins interact with water, provides elasticity. For flaky pastries, we want minimal gluten development to maintain tenderness. This is achieved by using cold ingredients, minimal handling, and a high fat-to-flour ratio. Fat shortens gluten strands, contributing to tenderness and creating layers (in laminated doughs like puff pastry) that expand with steam during baking.
